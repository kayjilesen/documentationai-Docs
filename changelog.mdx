<Update label="2024-10-15" description="v1.2.0" tags={["feature", "improvement"]}>

## New Features

- Added support for custom brand colors in documentation themes, starting with `#1A2D35` for Kay Jilesen branding.
- Introduced Mermaid diagram rendering for visualizing web design workflows and project strategies.

## Improvements

- Enhanced search functionality to better index WordPress specialist guides and case studies.
- Optimized page load times by `>30%` for documentation pages with heavy image mockups.

## Bug Fixes

- Fixed rendering issues with Dutch language content in service descriptions.
- Resolved anchor link navigation for sections like "Strategie.Design.Code."

</Update>

<Update label="2024-09-20" description="v1.1.0" tags={["feature", "bugfix"]}>

## Bug Fixes

## Improvements

</Update>

## How to Upgrade

Follow these steps to ensure your documentation stays current.

<Steps>
  <Step title="Check Version" icon="package">
    Verify your current version by running:
    
    ```
    npm list kay-jilesen-docs
    ```
  </Step>
  <Step title="Update Package" icon="download">
    <Tabs>
      <Tab title="npm" icon="package">
        ```
        npm update kay-jilesen-docs
        ```
      </Tab>
      <Tab title="yarn" icon="package">
        ```
        yarn upgrade kay-jilesen-docs
        ```
      </Tab>
    </Tabs>
  </Step>
  <Step title="Review Changes" icon="book-open">
    Scan this changelog for breaking changes and test your pages.
  </Step>
</Steps>
