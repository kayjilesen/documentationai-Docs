## Quick Start

1. Sign up at `https://kayjilesen.com/register`
2. Verify your email address
3. Log in to your dashboard
4. Create your first documentation project

## Editing Docs

- Select a project
- Click "New Page" to add content
- Use Markdown or rich text editor

## Share

- Use the Share button to generate a public link
